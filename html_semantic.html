Mene ne Semantic HTML? What is Semantic HTML?

Semantic HTML na nufin amfani da alamomi masu ma'ana don bayyana abin da ke cikin shafin. Wannan yana taimakawa masu nakowa da kuma injunan bincike (search engines). Misali, maimakon amfani da <div> a ko'ina, za mu iya amfani da <header>, <nav>, <main>, <article>, da <footer>.

Semantic HTML means using meaningful tags to describe the content. This helps screen readers and search engines understand your page. Instead of using <div> everywhere, we can use <header>, <nav>, <main>, <article>, and <footer>.

Misali / Example
<header>
  <h1>Taken Shafi</h1>
</header>

<main>
  <article>
    <h2>Labarinmu</h2>
    <p>Wannan shine abun ciki...</p>
  </article>
</main>

<footer>
  <p>© 2026 DigitalHausa</p>
</footer>

Alamomin Semantika:

  • <header>: Sashin kan shafi (logo, taken, nav). Top section of page.
  • <nav>: Hanyoyin shawagi (navigation links). Navigation links.
  • <main>: Babban abun cikin shafi. Main content area.
  • <article>: Mai zama abu mai cin kansa (blog post, news). Self-contained content.
  • <footer>: Sashin ƙasan shafi. Bottom section of page.
